## IRCTC Website Login Page Clone

Overview

This project is a frontend recreation of the IRCTC website's header and login section, designed for desktop and mobile screens. It features a clean, spacious layout with a modern look and mobile responsiveness. Additionally, it includes a banner section with gradient background and features highlighting IRCTC bus booking.

The project uses HTML, CSS, and JavaScript only, without any backend. It focuses on responsive design, card layouts, and interactive elements like password toggle and OTP mode.

## Features

Header

Clean, spacious design replicating the IRCTC top section.

Includes Indian Railways logo (left) and IRCTC logo (right).

Top section shows current date & time, font size controls, and Hindi/English toggle.

Includes home icon, login/register button, and navigation links.

Responsive: Header items wrap on smaller screens for mobile devices.

## Login Card

Modern login form with floating labels for inputs.

Password visibility toggle feature.

Option to enable OTP-based booking.

Footer with Register and Agent Login buttons.

Login card height matches banner card for a uniform layout.

## Banner Card

Light blue gradient background for a visually appealing banner.

Icon representing bus booking.

Title and subtitle with key features of IRCTC Bus Booking.

Decorative floating circles for enhanced aesthetics.

Equal height with login card for uniformity.

## Responsiveness

Cards stack vertically on mobile devices.

Header items wrap instead of overflowing.

Smooth, mobile-friendly layout that retains readability and accessibility.

## Interactivity (JavaScript)

Real-time date and time display.

Toggle password visibility feature.

Enable/disable password input when OTP mode is checked.

Simulated login button interaction with temporary processing animation.
