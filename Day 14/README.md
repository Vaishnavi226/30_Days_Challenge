## Responsive Payment Method UI

A modern and responsive **Payment Method Interface** built using pure **HTML and CSS**.  
The layout adapts beautifully for both **desktop and mobile screens** without requiring scrolling.

This project focuses on UI alignment, spacing optimization, responsiveness, and real-world layout practice inspired by e-commerce checkout pages.


## Features

- Wide desktop layout with compact height  
- Mobile responsive layout (phone-friendly view)  
- Clean UI with proper spacing and alignment  
- Flexible payment options layout  
- Card logos integration  
- No scrolling required on desktop  
- Pure HTML & CSS (No frameworks)


## Technologies Used

- HTML5  
- CSS3  
- Flexbox  
- Media Queries
