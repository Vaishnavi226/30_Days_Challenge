## IRCTC Train Booking UI Clone

## ProjectOverview

This project is a frontend UI clone of the IRCTC Train Search and Booking Results page, built using HTML, CSS, and JavaScript. It includes train listing cards, filters, class selection, and an interactive “Other Dates” modal.

## Features

IRCTC-style Navbar
Sidebar “Refine Results” Filters (Train Type, Departure Time, Arrival Time)
Train Results Section with 3 Train Cards
Train Class Selection with dynamic refresh and random availability
Auto price generation when availability is found
Book Now button enabled only after selecting an available class
Other Dates modal with next 8 days date generation
Random availability shown for dates
Selected date updates the train card
Reset Filters option to restore default state

## TechStack

HTML5
CSS3
JavaScript (DOM Manipulation and Event Handling)
Font Awesome Icons
