# IRCTC Next Generation eTicketing System – UI Clone

A responsive and interactive **IRCTC Next Generation eTicketing System UI Clone** built using **HTML, CSS, and JavaScript**.  
This project focuses on recreating the look, feel, and basic functionality of the official IRCTC website with clean UI, proper spacing, and interactive features.

## Features

- Modern and clean UI inspired by IRCTC
- Fully responsive layout for desktop screens
- Search trains functionality
- Dynamic train listing
- Class availability selection
- Time slot filters
- Sorting by departure time, duration, and price
- Interactive sidebar filters
- Improved fonts, spacing, and layout
- No page overflow or unnecessary scrolling

## Technologies Used

- **HTML5** – Structure of the webpage
- **CSS3** – Styling, layout, and UI improvements
- **JavaScript (Vanilla JS)** – Dynamic behavior and interactivity
