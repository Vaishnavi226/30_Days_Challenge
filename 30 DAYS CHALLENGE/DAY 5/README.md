# Overlay Card Design (Bootstrap 5)

This project demonstrates a modern **overlay card layout** where a white content card overlaps an image.  
It is built using **HTML**, **CSS**, and **Bootstrap 5 Starter Template**.

The layout is fully responsive and works smoothly on desktop, tablet, and mobile devices.

## Features

- Bootstrap 5 starter template
- Image aligned on the right side
- Content card overlapping the image
- Responsive design for mobile and tablets
- Clean UI with modern spacing
- Easy customization

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
