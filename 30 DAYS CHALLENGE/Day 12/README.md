# Stopwatch Web Application

This project is a simple and clean Stopwatch Web Application built using web technologies. It provides a user-friendly interface to measure time accurately with basic control options.


# Features

Displays time in Hours : Minutes : Seconds : Milliseconds

Start button to begin the stopwatch

Pause button to temporarily stop the time

Reset button to clear the time back to zero

Clean and minimal user interface



# Technologies Used

HTML – For structuring the stopwatch layout

CSS – For styling the user interface
