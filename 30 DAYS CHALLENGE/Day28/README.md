# IRCTC Tourism Clone Website

A front-end clone of the IRCTC Tourism package listing page built using HTML, CSS, and JavaScript.

This project replicates the UI structure of the IRCTC Tourism website including:

- Header navigation
- Carousel section
- Sidebar filters
- Package listing cards
- Pricing section
- Footer with links & social icons

## Technologies Used

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (DOM Manipulation)
- Font Awesome (Icons)
