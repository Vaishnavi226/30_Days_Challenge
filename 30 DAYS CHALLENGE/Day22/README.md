# IRCTC Hotels Clone (Frontend)

A responsive frontend clone of the IRCTC Hotels booking website, built using HTML, CSS, and JavaScript.  
This project includes a hotel search section, banner slider, services menu, popular destinations, FAQ accordion, app download section, footer, and interactive UI effects.

## Features

- Fixed header with IRCTC branding
- Hero section with background image
- Domestic / International toggle buttons
- Hotel search form (Destination, Dates, Rooms, Guests)
- Banner image slider with:
  - Next/Previous buttons
  - Auto-slide every 4 seconds
  - Clickable dots navigation
- Promo cards (Luxury, Budget, Beach, Hill Stations)
- Services icons menu (Hotels, Flights, Bus, Lounge, etc.)
- Popular destinations section with cards
- FAQ accordion (expand/collapse)
- App download section with:
  - Play Store and App Store badges
  - QR Code
  - Rating display
- Scroll-to-top button
- Smooth animations on scroll
- Fully responsive design for mobile and tablet

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Unsplash Images (for UI demo)
