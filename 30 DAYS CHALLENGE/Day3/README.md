Testimonial Card Page

A responsive testimonial card page built with HTML, CSS, and Bootstrap 5.
It showcases what users are saying about tools listed on the platform in a clean, modern, and mobile-friendly layout.

Features

Responsive design for all screen sizes

User logo, name, and Twitter handle displayed

Justified testimonial text for readability

Card footer with likes, comments, and timestamp

Proper spacing between cards for small and medium devices

Technologies Used

HTML5

CSS3

Bootstrap 5

Bootstrap Icons
