# IRCTC eCatering | Food on Track (UI Clone)

This is a front-end UI clone of the **IRCTC eCatering (Food on Track)** website.  
It is built using **HTML, CSS, and JavaScript** with a clean and responsive layout.

---

## Project Features

- Sticky Header with navigation links
- Hero section with background image and search/PNR input
- Info strip showing IRCTC official details
- Group Orders banner section
- "How it works" steps section
- Important stations grid with pagination UI
- Download App section with store buttons
- Vendor registration banner
- Footer with links and social icons

---

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (basic file included)**
- **Google Fonts (Roboto)**
- **Font Awesome Icons**
