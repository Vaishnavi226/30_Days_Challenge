# Kodee UI – AI Chat Interface

A modern, responsive chatbot UI called Kodee, built with HTML, CSS, and JavaScript. Users can interact via a chat popup, click on suggestions, or type messages in a mini input box over the image. Messages are dynamically displayed in the chat interface with simulated bot responses.

# Features

Responsive layout for desktop, tablet, and mobile screens.

Top-left image section with a sleek mini input box.

Chat popup that opens/closes with a button.

Dynamic user messages with styled chat bubbles.

Bot simulated responses.

Clickable suggestion cards to send pre-defined messages.

Press Enter to send a message.

Integrated mini input box sends text directly to the chat popup.

# Technologies Used

HTML5 – Structure and layout of the interface.

CSS3 – Styling, animations, and responsiveness.

JavaScript – Chat functionality, message handling, and interactivity.
