# IRCTC - Booked Ticket History (UI Clone)

This project is a frontend UI clone of the IRCTC "Booked Ticket History" page using **HTML, CSS, and JavaScript**.

## Features

- Clean and modern IRCTC-like layout
- Tabs UI (All Journeys, Upcoming, Past Journeys)
- Ticket Cards with:
  - Passenger information table
  - Transaction details
  - Ticket fare
  - PNR section
  - Action buttons
- Sidebar Advertisement Card
- Fully Responsive (Mobile + Tablet + Desktop)

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts (Roboto)
