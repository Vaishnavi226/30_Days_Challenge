# IRCTC Review Journey Page Clone

This project is a frontend UI clone of the **IRCTC "Review Journey" page**, built using **HTML, CSS, and JavaScript**.  
It includes a modern layout, stepper progress bar, passenger details toggle, fare summary sidebar, CAPTCHA generator, and toast notifications.

## Features

- Top bar with IRCTC branding and welcome info
- Navigation menu similar to IRCTC website
- Train journey card with departure, arrival, duration, quota badge
- Passenger details section with toggle (collapse/expand)
- Contact info section
- CAPTCHA generator with refresh button
- CAPTCHA validation before continuing
- Stepper progress update (Passenger → Review → Payment)
- Toast notifications for success/error/info
- Fully responsive UI (mobile + tablet supported)
- Sticky Fare Summary sidebar

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **Font Awesome Icons**
