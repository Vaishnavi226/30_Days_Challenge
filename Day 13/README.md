# Developer Profile Cards UI

This project is a responsive Developer Profile Card UI built using HTML and CSS.  
It displays software developer cards with profile images, role, name, description, and a follow button.

The cards are centered on the page and adapt smoothly across mobile, tablet, and desktop screens.
