## IRCTC Next Generation eTicketing System – UI Clone

This project is a frontend UI clone of the IRCTC Next Generation eTicketing System, built using HTML, CSS, and JavaScript.
The goal of this project is to closely replicate the real IRCTC homepage layout while following clean UI structure and responsive frontend practices.

## Project Overview

The IRCTC UI Clone recreates the booking interface, navigation bar, service sections, alert modal, and supporting UI elements seen on the official IRCTC website.
It focuses purely on UI/UX design and frontend behavior, not backend ticket booking logic.

## Technologies Used

HTML5 – Structure of the web page

CSS3 – Styling, layout, and responsiveness

JavaScript (Vanilla JS) – Interactivity and dynamic behavior

Font Awesome – Icons used across the UI

## Features Implemented

Top navigation bar with date-time and accessibility controls

IRCTC-style navigation menu with multiple services

Ticket booking card (From, To, Date, Class, Quota)

Swap functionality for From & To locations

Checkbox options for concessions and flexibility

Search Trains button UI

Services grid (Flights, Hotels, Catering, Bus, Holidays, etc.)

Fraud alert banner (IRCTC warning message)

Advertisement placeholder section

Floating “Ask Disha” chatbot UI

Modal popup alert with bilingual (English + Hindi) message
