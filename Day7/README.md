# Crunchyroll Clone - Responsive Web Page

## Overview

This project is a **clone of Crunchyroll's homepage hero section** built using **HTML and CSS**.  
It features a responsive **navbar**, **hero section with heading, paragraph, and buttons**, and is optimized for **desktop, tablet, and mobile screens**.

## Features

- **Responsive Navbar**

- **Hero Section**

- **Responsive Design**

## Technologies Used

- **HTML5**
- **CSS3 (Flexbox, Media Queries)**

## How to Use

1. Clone the repository or download the files.
2. Open `index.html` in any modern browser.
3. Resize the browser window or open on mobile to see responsive behavior.
