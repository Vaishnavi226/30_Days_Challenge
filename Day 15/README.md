# Order Confirmation Page UI

A responsive Order Confirmation webpage built using **HTML5 and CSS3**, inspired by real-world e-commerce confirmation layouts.  
This project focuses on clean layout structure, typography, responsive design, and component-based styling.

## Features

- Clean and professional e-commerce UI
- Responsive layout for desktop and mobile devices
- Flexbox-based layout structure
- Reusable and well-organized CSS
- Amazon-style button design
- Order summary and product details section
- Mobile-friendly responsive design using media queries

## Technologies Used

- **HTML5** – Page structure and content
- **CSS3** – Styling, layout, responsiveness
- **Flexbox** – Alignment and layout control
