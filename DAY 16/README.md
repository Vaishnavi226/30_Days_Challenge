## IRCTC Next Generation eTicketing System – UI Clone

A frontend UI clone of the IRCTC Next Generation eTicketing System, built using HTML, CSS, and JavaScript.
This project focuses on recreating the real IRCTC homepage layout, booking card, services section, and overall visual design.

## Project Overview

This project replicates the IRCTC train ticket booking interface, including:

Header navigation with official branding

Hero banner with booking widget

Train booking form (From, To, Date, Class, Quota)

Service shortcuts (Flights, Hotels, Bus, Catering, etc.)

Fraud awareness banner

Floating chatbot (Ask Disha)

The goal of this project is UI accuracy, layout structuring, and frontend best practices.

## Tech Stack

HTML5 – Page structure

CSS3 – Styling, layout, flexbox, hover effects

JavaScript (Vanilla JS) – Basic interactivity

Font Awesome – Icons
