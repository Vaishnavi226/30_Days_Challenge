## IRCTC Bus Ticket Booking UI Clone

A responsive frontend clone of the IRCTC Bus Ticket Booking Website built using HTML, CSS, and JavaScript. This project focuses on recreating the user interface, animations, and interactive components inspired by the official IRCTC bus booking platform.

## Project Overview

This project replicates the IRCTC Bus booking homepage design including:
Hero banner with animated buses
Service navigation icons
Bus search booking section
Trending packages section
FAQ accordion UI
App download promotional section
Fully designed footer
Custom date picker functionality
Smooth animations and hover effects

The project demonstrates strong frontend development skills including layout structuring, DOM manipulation, and UI animations.
